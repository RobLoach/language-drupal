# Drupal support in Atom

[Drupal](http://drupal.org) syntax highlighting and snippets for Atom.


## Install

1. File -> Preferences...
2. Click on **Packages**
3. Search for *"Language Drupal"*
4. Click **Install**


## Sources

* [Drupal Sublime Snippets](https://github.com/juhasz/drupal_sublime-snippets)
